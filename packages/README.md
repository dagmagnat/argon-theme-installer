# Packages / Пакеты

This directory contains bundled Argon packages used by `install.sh`.

Эта папка содержит пакеты Argon, которые использует `install.sh`.

## OPKG

Used on classic OpenWrt/OPKG-based systems.

Используется на классических OpenWrt/OPKG-системах.

```text
packages/opkg/luci-theme-argon_2.3.2-r20250207_all1.ipk
packages/opkg/luci-app-argon-config_0.9_all.ipk
```

## APK

Used on newer OpenWrt/APK-based systems.

Используется на новых OpenWrt/APK-системах.

```text
packages/apk/luci-theme-argon-2.4.3-r20250722.apk
packages/apk/luci-app-argon-config-1.0-r20230608.apk
```

Note: APK here means OpenWrt/Alpine package format, not Android APK.

Примечание: APK здесь означает пакетный формат OpenWrt/Alpine, а не Android APK.
